Element Hide/Show

Helpful Method: hide(), show(), toggle() fade(), slide()

Example 1: hide()/show()

HTML Code

// for show action


//for hide action




	Test

jQuery Code

// use show() method
$('#show').click(function(){
	$('.block').show();	
});

// use hide() method
$('#hide').click(function(){
	$('.block').hide();
});

Result

CSS Code

.block{
	display: none;
}
In above example, we have used two seperate buttons to understand the hide/show action. But mostly person want to use only one button for performing hide/show feature. We can do the work by help help of toggle() method. In next example we will explain toggle() method.

Example 2: toggle()

HTML Code

// for hide/show action




	Test

jQuery Code

// use toggle() method
$('#toggle').click(function(){
	$('.block').toggle();	
});

Result

CSS Code

.block{
	display: none;
}
In above example, we have used only one button to hide/show. toggle() method check the selecter css property("display"). if selecter have "display:none" than toggle() method performing as a show() method and if selecter have "display: block" or not have display css property that toggle() method performing as a hide().

//for hide action




	Test

jQuery Code

	$('.block').fadeIn();	
});

// use fadeOut() method

	$('.block').fadeOut();
});

Result

CSS Code

.block{
	display: none;
}

In above example,
fadeIn(): work same as show() method
fadeOut(): work same as hide() method
But, difference is fadeIn/fadeOut perform with fading effect.

we have used two seperate buttons to understand the fadeIn/fadeOut action. In jQuery not available a common method for fadeIn/fadeOut like toggle(). If need to use only one button to fadeIn/fadeOut than need to do on condition based. In next example we will explain it.

Example 4: fade toggle with fadeIn()/fadeOut()

HTML Code

// for hide/show action




	Test

jQuery Code

// use fadeIn/fadeOut method
$('#toggle').click(function(){
	if($('.block').css('display') == 'none'){
		$('.block').fadeIn();
	}else{
		$('.block').fadeOut();
	}
});

Result

CSS Code

.block{
	display: none;
}
In above example, we have used only one button to hide/show. On click, function check the selecter css property("display"). if selecter have "display:none" than fadeIn() method performing and if selecter have "display: block" or not have display css property that fadeOut() method performing
