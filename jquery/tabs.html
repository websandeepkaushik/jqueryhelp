jQuery Code

$('.tab a').click(function(){
	var obj = $(this).parents('.tab-section');
	obj.find('.tab li').removeClass('active');
	obj.find('.tab-content').hide();

	$(this).parent('li').addClass('active');
	obj.find($(this).attr('href')).show();
	return false;
});

$('.tab li:first-child a').trigger('click');

Result

CSS Code

.tab-section{
	max-width: 500px;
}
.tab{
	margin: 0;
	padding: 0;
	list-style: none;
}
.tab li{
	display: inline-block;
}
.tab a{
	display: block;
	padding: 10px;
	background: #DDD;
	color: #333;
	text-decoration: none;
}
.tab li.active a{
	background: #CCC;
}
.tab-content{
	border: 1px solid #CCC;
	padding: 10px;
	display: none;
}

In above example, we have used hide() and show() method to understand the tabbing functionality. In this example two action perform that are necessary for tabs, First hide the all ".tab-content" section. Second show the particular selected single ".tab-content" that you want to display. For get the selected ".tab-content" object we will use the "attr" method, in example we are getting "href" value and use it as a selected object and trigger the show() method.

In this example we have used one more action for highlight the current tab, for do this action used addClass() method. We need to add two lines of code, first remove the highlighted tab this work will complete with removeClass() method and second show the highlighted tab this work will complete with addClass() method.
